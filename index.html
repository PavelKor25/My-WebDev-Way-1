Текущее состояние

На данный момент я прошел первые этапы обучения веб-разработки, пройдясь по основным тегам и атрибутам языка HTML. Данную страницу я символически не буду оформлять далее тех знаний, которыми я обладаю в момент написания (т.е. без CSS и JavaScripts) Пусть это будет ретро-вариант :).

На данный момент я умею создавать заглавия, параграфы, горизонтальные линии, перенос строки, гиперссылки, выкладывать изображения и т. д. Также я разбираюсь с тем, как пользоваться Relative Path (букв. "Относительный путь к файлу"). В конце я научился хостить веб-сайты через GitHub.

Как это было

На самом деле, HTML - очень простой язык. Основная его суть заключается в знаниях тегов и атрибутов к ним. Приятно также, что он интуитивен и обладает простыми параметрами без излишек. Достаточно воспользоваться данной документацией по html для понимания ее основных аспектов. Конечно же, все же имеются другие важные аспекты при создании веб-страниц. К примеру:

  • HTML Boilerplate
  • Специальная программа Visual Studio Code и ее особенности
  • Lorem Ipsum (полезная штука на самом деле)
  • Конфиденциальность своей страницы
  • И еще я что-то мог забыть :)

В конце я бы рекомендовал посетить вот этот сайт как исторический памятник всех моих начал :)
